This app requires you to have a Virtual Audio Cable. Please install one depending on your operating system.

This app currently only works with the following cables. If you are using something else, it won't work.

Windows

Muzychenko VAC works in Windows XP, Vista, 7, 8, 8.1 and 10.

Download the free version HERE.

MacOS

VB-Audio works in MacOS.

Download the free version HERE.

After installing, please go to your sound settings -> microphone, and make sure that the audio level for the VB-Cable microphone is full volume.
